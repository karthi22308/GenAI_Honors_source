Steps to run the application

1. Clone to local
2. inside the folder open the command prompt
3. install the required pacakges by running:
   pip install -r requirements.txt
4. Replace the Valid Key in .env File
5. run the streamlit application by command:
   streamlit run app.py
6. Application can be tested with any valid pdf document
7. for submission zip the app.py file only without .env file
